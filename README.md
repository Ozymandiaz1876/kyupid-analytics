# analytics-kyupid

## Project setup
```
npm install
```

create a .env file and add following data in it

Base url for the apis
```
VUE_APP_API_URI = https://kyupid-api.vercel.app/api
```
Mapbox access token

```
VUE_APP_MAP_ACCESS_TOKEN = MAPBOX_ACCESS_TOKEN
```
